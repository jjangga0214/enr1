# How to use?
+++
document title
===
```markdown
document title
===
```
+++
medium title
-----
```markdown
medium title
-----
```
+++
### small title H3
#### small title H4
<h5 style="font-size:0.7em !important"> small title H5</h5>
<h6 style="font-size:0.5em !important"> small title H6</h6>
```markdown
### small title H3
#### small title H4
##### small title H5
###### small title H6
```

+++
### List
* higher element
  * middle element
    * lower element
* higher element

```markdown
* higher element
  * middle element
    * lower element
* higher element
```

### Code
```python
name  = input("What is your name? : ")
print("Hello, %s" %name)
```

```markdown
'''python
name  = input("What is your name? : ")
print("Hello, %s" %name)
'''
```
